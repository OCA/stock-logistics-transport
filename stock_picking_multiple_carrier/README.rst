================================
Stock Picking - Multiple Carrier
================================

* Each picking can have more than one Carrier
* The user can select the carrier type and assign a tracking number for it

Installation
============

* No thing is required to install this module
* You can install the migration module "migration_stock_picking_multiple_carrier" (Optional)
